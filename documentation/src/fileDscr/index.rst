File Description `(fileDscr) <https://docs.ddialliance.org/DDI-Codebook/2.5/xmlschema/schemas/codebook_xsd/elements/fileDscr.html>`_ provides information about the data file(s) that comprises a collection. This section can be repeated for collections with multiple files.

For each add a purpose

- fileTxt (0-n)
- fileDerivation (0-1)
- locMap (0-1)
- notes (0-n) [type, subject, level, resp, sdatrefs, parent, sameNote]
